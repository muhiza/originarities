Markdown Help

Contents that are composed and written for the 2buntu blog are written in a text-formatting language known as "Markdown". The language is both easy to read and easy to learn. What follows is a relatively brief overview of Markdown syntax and some examples that demonstrate its usage.

Simple Text Formatting

**bold** | *italic* | _also italic_

The Markdown above produces the results below when rendered:

bold | italic | also italic

You can create headings with the following syntax:

# Top Level Heading
## Second Level Heading

An alternative syntax is also supported:

Heading
=======

Images

Embedding images in posts is very simple:

![hover text](http://example.com/image.jpg)

The "hover text" is displayed when the user hovers the cursor over the image.

Note: if you are uploading a screenshot or image that you have permission to use, please take advantage of the upload image button in the toolbar.

Lists

In order to display an ordered or unordered list, prefix each item with the approriate bullet. For ordered lists, a period may be used after the numeral: 

1. Item 1
    1. Subitem 1
    2. Subitem 2
2. Item 2

An unordered list is created in a nearly identical manner:

* An item
* Another item
* Yet another item

Code

If your Content contains code or commands to be entered into a terminal, indent the block with four spaces: 

// Create code blocks like this one
// by indenting the code with four spaces

If you need to refer to a variable or identifier within a paragraph of normal text, use the backtick character (`): 

Use `printf()` to output text.

Alerts

You can insert an alert by using one of the alert blocks: 

[warning]This is a warning.[/warning]

This will result in the following block:

This is a warning.

Types of Alerts

There are three different types of alerts. They are listed below:

[info][/info] provides supplementary information of trivial importance
[warning][/warning] warns the reader of a potential hazard
[danger][/danger] serves as a strong caution against doing something potentially dangerous
